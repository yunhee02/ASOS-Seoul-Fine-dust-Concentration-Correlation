# 🌫️ 서울 미세먼지 분석 프로젝트

## 📖 프로젝트 개요
서울 지역의 미세먼지 및 초미세먼지 데이터를 분석하여  
시간·환경 요인에 따른 변화 패턴과 상관관계를 파악한 데이터 분석 프로젝트입니다.

---

## 🎯 분석 목적
- 미세먼지(PM10) 및 초미세먼지(PM2.5) 농도 변화 분석
- 기상 요인(온도, 습도 등)과의 관계 확인
- 시간대 및 계절별 패턴 파악

---

## 🛠️ 사용 기술
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

---

## 📊 주요 분석 내용
- 미세먼지/초미세먼지 농도 분포 분석
- 시간대 및 날짜별 변화 추이 분석
- 기상 데이터와의 상관관계 분석
- 시각화를 통한 패턴 도출

---

## 💡 결과 및 인사이트
- 특정 계절 및 시간대에서 미세먼지 농도 증가 패턴 확인
- 기온 및 환경 요인과의 일부 상관관계 확인
- 데이터 기반 환경 변화 이해 가능성 확인
